# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, professional, and respectful environment for all contributors, regardless of age, experience, identity, or background.

## Our Standards

To ensure a positive experience for everyone, we expect participants to behave respectfully, collaboratively, and constructively. Disrespectful or harmful behavior will not be tolerated.

## Enforcement Responsibility

Project maintainers are responsible for enforcing this Code of Conduct fairly and consistently.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, emails, chats, and any other related communication channels.

## Enforcement

When a behavior that violates this Code of Conduct is reported, maintainers will take appropriate actions, including but not limited to:
- Requesting behavioral changes
- Temporary or permanent removal from project access

## Enforcement Guidelines

1. Reports can be sent to maintainers via email or a private issue.  
2. Each report will be handled discreetly and with respect for the privacy of all involved parties.  
3. Enforcement actions will be proportional to the severity of the violation.

## Attribution

This Code of Conduct is based on standard open source community examples and adapted for this project.
